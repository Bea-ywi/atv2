Bem-vindo ao nosso guia de HTML!

Aqui você vai aprender de forma fácil e prática como utilizar as principais tags HTML para criar suas próprias páginas da web. Descubra a função de cada elemento e comece a dar vida às suas ideias online!

O que são tags no HTML?

Tags HTML são instruções que informam ao navegador como formatar o texto. Você pode usar tags para definir itálico, quebras de linha, objetos, bullet points e muito mais.
Essas tags estão presentes no HTML (Hypertext Markup Language) de todas as páginas da internet.

Qual é a diferença entre elementos, atributos e tags HTML?

Um elemento HTMLé um item da página — ele representa o próprio conteúdo

Uma tag HTML define como esse elemento será exibido ou estruturado.

Um atributo HTML adiciona informações extras que modificam ou detalham esse elemento.

Veja como eles funcionam juntos:

Conceito Descrição Exemplo
Tag São os comandos que envolvem um elemento, começando com < e terminando com >. <b>Texto em negrito</b>
Elemento É o conteúdo dentro dessas tags. Texto em negrito no trecho acima.
Atributo Dão características adicionais às tags, como idioma ou estilo. <html lang="pt-BR">

Regra da Cascata

A cascata determina qual regra CSS será aplicada quando houver mais de uma regra para o mesmo elemento.

A prioridade é definida por três fatores:

  1. Origem do estilo (user-agent, usuário, desenvolvedor)
  2. Especificidade do seletor
  3. Ordem de aparição (a última regra sobrescreve as anteriores se tiver a mesma especificidade)

Diferenças entre display inline e display block

  • Block: Ocupa toda a largura disponível, quebra linha antes e depois. Aceita width, height, margin, padding.
  • Inline: Ocupa apenas o tamanho do conteúdo, não quebra linha. Não aceita width e height diretamente, mas aceita margin-left/right e padding-left/right.

Elementos block por padrão: div, p, h1–h6, section, article, header, footer, main, nav.

Elementos inline por padrão: span, a, strong, em, img, label

Margin, Padding e Border

Margin

A margin (margem) é o espaço externo de um elemento. Ela controla a distância entre o elemento e outros elementos ao redor dele.

  • Exemplo: se você definir margin: 20px;, o elemento ficará afastado 20px de outros elementos.

Padding

O padding (preenchimento) é o espaço interno do elemento, entre o conteúdo e a borda (border). Ele empurra o conteúdo para dentro, criando espaço interno.

  • Exemplo: se você definir padding: 15px;, o texto dentro da caixa ficará afastado 15px da borda.

Border

A border (borda) é a linha que envolve o elemento, entre o padding e a margin. Ela pode ter espessura, cor e estilo.

  • Exemplo: border: 2px solid black; cria uma borda preta, sólida, de 2px de espessura.

Shorthand

É uma forma abreviada de escrever várias propriedades relacionadas em uma única linha, em vez de defini-las separadamente. Isso torna o código mais limpo, curto e fácil de ler.

  • margin: 10px; → aplica para todos os lados.
  • margin: 10px 20px; → 10px top/bottom e 20px left/right.
  • margin: 10px 20px 30px; → 10px top, 20px left/right, 30px bottom.
  • margin: 10px 20px 30px 40px; → top, right, bottom, left.

box-sizing: border-box

Por padrão (content-box), o width inclui apenas o conteúdo. Com box-sizing: border-box, o width passa a incluir conteúdo + padding + border, facilitando o controle do layout.
