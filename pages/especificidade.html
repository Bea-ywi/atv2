O que é especificidade?

  • Página principal

A especificidade no CSS define qual regra tem mais prioridade quando várias estilizações tentam modificar o mesmo elemento. Em outras palavras, ela determina qual estilo "vence" o conflito e será aplicado.

Como funciona?

Cada tipo de seletor possui um nível diferente de importância:

  • Seletor por tag (p, h1, div) → baixa especificidade.
  • Seletor por classe (.meu-texto) → especificidade média.
  • Seletor por ID (#titulo) → alta especificidade.

Pontuação da especificidade

A especificidade funciona como um sistema de pontos:

  • Tag: 1 ponto
  • Classe: 10 pontos
  • ID: 100 pontos

Combinação de seletores

  • Espaço (descendente): .primeiro h1 → Seleciona todos os h1 que estão dentro de .primeiro, não importa o nível.
  • Filho direto: .primeiro > h1 → Seleciona apenas h1 que são filhos diretos de .primeiro.
  • Irmão adjacente (soma): .primeiro h1 + h1 → Seleciona apenas o h1 que vem logo depois do primeiro h1 dentro de .primeiro.

Pseudo-classes

  • :first-child → Seleciona o primeiro filho de um elemento pai.
  • :last-child → Seleciona o último filho.
  • :nth-child(n) → Seleciona o filho pela ordem (ex.: nth-child(2) pega o segundo filho).
  • :nth-of-type(n) → Semelhante ao nth-child, mas considera apenas filhos de um tipo específico (ex.: só p ou só li).
